# DevastMod Browser Extension

This is the browser extension packaging for DevastMod. By installing this extension, you can inject all mod features directly into the official `devast.io` website over HTTPS without needing to self-host the client.

## Features

- **Global Chat Overlay**: Adds a custom global chat interface to communicate with other players using the mod.
- **Ad Removal**: Deletes advertisement containers and tracking scripts from the main game page.
- **Custom Skins Menu**: Enables picking custom skins (e.g., Fury, Devil, Athlete skins) that other mod users see seamlessly.

## Installation

This extension uses **Manifest V3** and can be loaded in Chromium-based browsers (Google Chrome, Microsoft Edge, Brave, etc.).

1. Open your browser and navigate to the Extensions page:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
2. Enable **Developer Mode** (usually a toggle switch in the top right corner).
3. Click the **"Load unpacked"** button.
4. Select this directory (`devastmod-extension-main`).
5. Open `https://devast.io` and enjoy the mod!
